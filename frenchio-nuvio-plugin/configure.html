P2P dans Nuvio
Pour que le streaming fonctionne, activez Paramètres → Lecture → Autoriser les streams P2P dans l'application Nuvio. Le plugin retourne des liens magnet ; c'est le moteur natif de Nuvio qui gère le téléchargement P2P.
✅ URL générée — Copiez-la dans Nuvio

Comment installer dans Nuvio

  1. Copiez l'URL ci-dessus
  2. Ouvrez Nuvio → Paramètres → Plugins
  3. Appuyez sur Ajouter un plugin
  4. Collez l'URL et confirmez
  5. Activez les streams P2P dans Paramètres → Lecture
