Bienvenue sur LeFlux.

                
LeFlux. est une solution portable d'agrégation de flux de streaming torrents francophones. Elle combine un addon Stremio/Nuvio, des scrapers JavaScript et des clients locaux (qBittorrent, TorrServer, JOAL) pour un streaming fluide, rapide et sécurisé.

                
Premiers pas :

                
    
                    
  1. Dans Configuration, renseignez vos clés API trackers (C411, Torr9, Gemini, TR4KER, YGG…). La clé TMDb est optionnelle.
    2. 
                    
  2. Configurez le VPN WireGuard si vous téléchargez des torrents en local (recommandé).
    3. 
                    
  3. Copiez le lien Addon depuis l'onglet Lecteur et installez-le dans Nuvio via Plugins → Ajouter URL.
    4. 
                    
  4. Activez/désactivez les modules dans Configuration selon vos besoins.
    5. 
                

            

            

                
Addon vs Plugin P2P LeFlux.

                
LeFlux. propose deux architectures pour s'adapter à tous vos besoins :

                

                    
FonctionnalitéLeFlux. Addon (Frenchio)LeFlux. P2P Plugin

                    
                        
FonctionnementServeur centralisé Python. Nuvio envoie la requête → le serveur cherche sur les trackers → renvoie le flux.Script JS s'exécutant dans Nuvio. Pas de serveur requis pour la recherche.

                        
Streaming LocalVia qBittorrent ou TorrServer sur le serveur d'hébergement.Via le moteur torrent natif de Nuvio sur l'appareil de lecture.

                        
CompatibilitéOptimale — TV Android, Smart TV, navigateurs, mobiles.PC, Android TV avec support P2P natif.

                        
Debrid & CacheRealDebrid, TorBox, AllDebrid. Zéro bande passante locale si debrid actif.P2P direct pur. Bonne bande passante requise.

                    
                

            

            

                
Sécuriser ses téléchargements avec WireGuard

                
Pour éviter tout problème avec les autorités de régulation (ex: Arcom en France), faites passer le trafic P2P par un VPN chiffré.

                
Marche à suivre :

                
    
                    
  1. Obtenez un fichier de configuration WireGuard (.conf) chez votre fournisseur VPN (Mullvad, ProtonVPN, Surfshark…).
    2. 
                    
  2. Allez sur Configuration → VPN WireGuard.
    3. 
                    
  3. Renseignez la clé privée, l'adresse assignée, l'IP endpoint et sa clé publique.
    4. 
                    
  4. Choisissez le mode de redirection : qBittorrent & TorrServer ou Toute la stack.
    5. 
                    
  5. Enregistrez. Le conteneur frenchio-vpn démarre avec un Kill Switch activé par défaut.
    6. 
                

            

            

                
Maintenir son Ratio avec JOAL

                
Les trackers privés francophones exigent un ratio Upload/Download ≥ 1.0. Un ratio trop bas vous prive du droit de télécharger.

                
JOAL (Just Another Leecher) simule de l'upload sur vos torrents de façon réaliste, sans consommer de bande passante réelle.

                
Utilisation :

                
    
                    
  1. Activez JOAL dans Configuration → Modules & Services.
    2. 
                    
  2. Accédez à l'interface via le diagramme ou Flux → JOAL → Interface.
    3. 
                    
  3. Glissez vos fichiers .torrent dans l'interface.
    4. 
                    
  4. Activez le VPN WireGuard pour JOAL si votre IP résidentielle est filtrée.
    5.
